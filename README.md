# Challenge_3_DS
Este repositorio contiene un análisis exploratorio de datos (ETL) sobre la evasión de clientes (Churn) en TelecomX, una empresa de telecomunicaciones en LATAM. El objetivo es identificar patrones y factores que influyen en la retención de clientes, utilizando datos demográficos, de servicios y financieros.

Problema: La evasión de clientes representa una pérdida significativa de ingresos. Este análisis ayuda a entender por qué los clientes abandonan (e.g., contratos mensuales, altos cargos) y propone recomendaciones para reducirla.
Datos: Basado en un archivo JSON con ~7,043 registros de clientes, incluyendo variables como tenure, Contract, PaymentMethod y Charges.Monthly/Total.
Herramientas: Python con Pandas, Matplotlib y JSON para ETL y visualizaciones.
Insights clave: Churn ~27%, mayor en contratos "Month-to-month" (43%) y pagos "Electronic check" (~45%). Clientes leales generan más revenue.

El análisis se realiza en un notebook Jupyter, con secciones de extracción, transformación, carga, EDA y un informe final.
